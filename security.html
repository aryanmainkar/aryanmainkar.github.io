Scope & authorization

The live scan only ever targets aryanmainkar.github.io — a site I own, where I have explicit authorization to scan. The "try the demo" mode is a pre-rendered example, not a real scan, so paste-any-URL never triggers an outbound request to a third-party site.

That's deliberate. Unauthorized scanning of someone else's infrastructure is legally and ethically off-limits, even with passive checks. If you'd like a real scan of a site you own, get in touch — happy to run it under a written engagement.
