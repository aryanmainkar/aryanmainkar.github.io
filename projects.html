BroadAxis Inc.

Owning the UI/UX and front-end pipeline for 20+ production sites, with AI-driven creative workflows folded directly into delivery.

Problem

20+ client sites and a 60,000+ product catalog needed consistent UI patterns and a Year/Make/Model mapping that didn't fall apart at scale. Content production was bottlenecked on external designers and one-off layouts.

Approach

Built a reusable component library and custom WordPress plugins to standardize UI and handle YMM mapping. Folded Midjourney, Firefly, and Google Flow into the content pipeline so on-brand imagery shipped without external dependencies.

Outcome

Pixel-perfect, responsive, cross-browser delivery across the portfolio. Content turnaround dropped sharply, mapping accuracy held across the catalog, and the component library now backs every new client engagement.

LSB Steganography Application.

Hide and retrieve messages inside images using least-significant-bit encoding — full-stack build with a clean upload/preview UX.

Problem

LSB steganography is straightforward in theory but brittle in practice — image format conversion silently destroys hidden bits, and most tutorial implementations skip the user-facing flow entirely.

Approach

Designed an upload and preview flow in clean responsive HTML/CSS, then built the encoding and retrieval pipeline end-to-end in Flask. Persistence on Firestore, hosted on GCP.

Outcome

Working full-stack web app — message embedding and retrieval round-trips correctly on PNG inputs, with a UI that doesn't make you think about pixel manipulation.
